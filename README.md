## 💡 What is ChatJudge?
An AI-powered WhatsApp chat analyzer that reads 
your conversations and gives brutally honest, 
logical, evidence-backed personality reports 
for each participant.


## 🌟 Why ChatJudge?

We've all been there — a conversation that left us 
wondering "wait, was that toxic?" or "am I 
overreacting?" or "who actually started this mess?"

ChatJudge removes the guesswork.

Instead of relying on gut feeling, ChatJudge uses 
AI to read every single message, identify behavioral 
patterns, and deliver a structured, evidence-backed 
report — with actual quoted lines as proof.

No assumptions. No bias. Just the chat, analyzed.

---

## 💪 Key Benefits

### 🔍 Evidence-Based Analysis
Every score is backed by real quotes from the chat.
The AI doesn't just give numbers — it tells you 
exactly which messages led to that verdict and why.
No random scoring. Pure logic.

### 👥 Multi-Person Support
Analyzes every participant in the conversation 
individually. Whether it's a one-on-one chat or 
a group conversation — everyone gets their own 
detailed report card.

### 🚩 Pattern Detection
Single messages can be misleading.
ChatJudge looks at the overall pattern — who 
consistently dominates, who always deflects, 
who never apologizes — and scores accordingly.

### 😂 Actually Fun to Use
Brutal verdict labels, roast mode, compatibility 
scores, red flag counters — ChatJudge makes 
self-reflection genuinely entertaining.
Share it with friends. Send it to your situationship.

### 🔒 100% Private
Your chat never leaves your browser.
No data is stored, logged, or sent to any server 
other than the AI API for analysis.
What happens in the chat, stays in the chat.

### ⚡ Instant Results
Paste your chat, hit analyze, get your report 
in seconds. No signup. No waiting. No nonsense.

## ✨ Features

### 🧠 Core Analysis
- 8 trait scoring per person (0-100):
  - ☠️ Toxicity
  - 👑 Ego
  - 😤 Attitude
  - ❤️ Love
  - 😡 Hate
  - 😂 Humor
  - 🗣️ Dominance
  - 🧊 Coldness
- Funny AI-generated verdict labels
- Actual quoted lines as proof
- Logical explanation for every score
- Radar chart per person (visual comparison)

### 🚩 Flag System
- Red flag counter 🚩 with listed examples
- Green flag counter 💚 with listed examples

### 😂 Fun Features
- Roast Mode — AI savage roast based on behavior
- Compatibility score between participants
- Who should apologize first? — AI decides logically
- Relationship prediction — "These two won't last 💀"
- Chat personality type (MBTI style label)

### 📊 Chat Analytics
- Message count per person
- Average response time tracker
- Most used words per person
- Emoji usage analysis
- Conversation starter tracker
- Tone timeline — mood shift over time
- Manipulation detector (gaslighting, guilt tripping)
- Apology authenticity checker

### 📤 Shareable Output
- Download full report as PDF
- Share card — Instagram/WhatsApp size image
- One-click verdict copy

### 🔒 Privacy
- Name anonymizer (Person A / Person B mode)
- Chat is never stored or sent to any server
- Dark / Light mode toggle
- Language selector — Hindi / English output

### 💬 Overall Report
- Relationship health score (0-100)
- Label — Toxic / Complicated / Manageable / Healthy
- 3-4 sentence honest relationship assessment

## 🚀 How to Use

1. Export your WhatsApp chat
   > Chat → 3 dots → More → Export Chat → Without Media
2. Open ChatJudge
3. Paste the exported chat text
4. Click **Analyze**
5. Get exposed 💀
